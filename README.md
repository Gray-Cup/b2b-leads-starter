# B2B Leads Starter

A modular, security-oriented B2B template with admin panel and storefront.

**Free template by [GrayCup](https://graycup.org)**

## Quick Start

```bash
pnpm install
pnpm dev
```

## Apps

| App | Port | Description |
|-----|------|-------------|
| Storefront | 3000 | Customer-facing store |
| Admin | 3001 | Lead management panel |

## Setup

1. Run `supabase/schema.sql` in your Supabase SQL Editor
2. Configure environment variables (see [usage.md](usage.md))
3. Customize branding in `apps/*/lib/site.config.ts`
